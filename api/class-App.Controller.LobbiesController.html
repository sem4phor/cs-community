# isAuthorized( array|null $user )

isAuthorized

isAuthorized

Check if the provided user is authorized for the request: Banned users arent authorized for any action. All registered users can create one lobby. Only lobby-owners can kick users from their lobby. You can only delete your own lobby. You cant leave your own lobby. You can join a lobby if you arent already part of a lobby and have the required playtime.

Parameters

$user
The user to check the authorization of. If empty the user in the session will be used.

Returns

boolean
True if $user is authorized, otherwise false
