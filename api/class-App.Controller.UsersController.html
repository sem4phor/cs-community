isAuthorized

isAuthorized

Check if the provided user is authorized for the request. If the user is banned (role_id = 4) he is not authorized for anything. Moderators are allowed to ban and unban users, if they arent mods or admin.

Parameters

$user
The user to check the authorization of. If empty the user in the session will be used.

Returns

boolean
True if $user is authorized, otherwise false
